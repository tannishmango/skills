# skills

A collection of Agent Skills for [Cursor](https://cursor.com). Each skill gives the AI agent specialized knowledge, workflows, and capabilities it wouldn't reliably have on its own.

## Skills

### `give-me-a-prompt`
Generates a structured, copy-pasteable handoff prompt when a session ends and work continues in a new chat. Runs a self-audit to surface silent omissions, incomplete work, and gaps before handing off — so the next agent starts with the full picture, not a false one.

**Invoke:** *"give me a prompt"*, *"handoff"*, *"continue in new chat"*, *"next agent"*

---

### `give-me-a-walkthrough`
Generates a self-contained interactive HTML walkthrough for any skill, workflow, or concept. Opens directly in the browser — multi-section guided tour with navigation, progress tracking, and interactive exercises. Every output is designed from scratch for the subject.

**Invoke:** *"give me a walkthrough"*, *"tour me through X"*, *"show me how X works"*

→ **[See a live demo output](demos/give-me-a-walkthrough/give-me-a-prompt-walkthrough.html)** — a full interactive walkthrough of the `give-me-a-prompt` skill, generated by this skill.

---

### `skill-authoring`
Guides creation of high-quality Agent Skills using progressive disclosure, evaluation-first workflows, and verification loops. Covers structure, discovery metadata, runtime/tooling requirements, and the observe-refine-test loop.

**Invoke:** *"create a skill"*, *"update this skill"*, *"write a SKILL.md"*

---

### `goal`
Defines rigorous outcome specifications for complex tasks with evidence-grounded success criteria and explicit anti-goals. Mitigates Goodhart's Law by separating what you want from how you'll measure it.

**Invoke:** *"define a goal"*, *"set success criteria"*, *"what does done look like"*

---

### `todo`
Manages project TODOs in `docs/todos/`. Add, list, promote, complete, break down, and implement tasks with structured file-based tracking.

**Invoke:** *"add a todo"*, *"what's on my todo list"*, *"break this down into tasks"*

---

## Installation

Clone the repo and run the install script. It symlinks each skill into `~/.cursor/skills/` so the repo is the single source of truth — edits are immediately live, and `git pull` updates everything.

```bash
git clone https://github.com/tannishmango/skills.git ~/Desktop/code/skills
cd ~/Desktop/code/skills
./install.sh
```

The script is safe to re-run. It skips skills already correctly linked and won't touch marketplace-installed skills (real directories).

### Updating

```bash
cd ~/Desktop/code/skills
git pull
```

That's it. Because skills are symlinked, the pull immediately takes effect — no re-running the install script needed.
